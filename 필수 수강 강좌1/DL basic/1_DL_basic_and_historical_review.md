### 인공지능

: 사람의 지능을 모방한 것

AI > ML > DL



#### key componenets

: data(자연어, 이미지...), model, loss, algorithm



- data: 데이터는 풀고자하는 문제의 타입에 의존함
- Model: AlexNet, GoogleNet, LSTM, GAN....
- Loss: 모델을 어떻게 학습할지, 기준치가 됨!
  - Regression Task: MSE
  - Classification Task: CE
  - Probabilistic Task: MLE
- Algorithm: ex) k-fold validation
